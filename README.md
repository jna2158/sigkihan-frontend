## 프로젝트 소개

스마트한 냉장고 관리를 도와주는 모바일 웹 서비스입니다.
식품을 등록하여 유통기한을 추적하고 관리하여 음식물 낭비를 줄이고 효율적인 식품 관리를 가능하게 합니다.

## 주요 기능

- 🔐 카카오 소셜 로그인
- 📝 식품 등록/수정/삭제
- 🔔 유통기한 임박 알림
- 📊 식품 소비/폐기 관리
- 👤 프로필 관리

### 필수 조건

- Node.js 16.x 이상
- npm 또는 yarn

### 설치

```bash
npm install
npm start
```
